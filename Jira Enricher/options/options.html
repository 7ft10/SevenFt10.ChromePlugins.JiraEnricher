Jira Enricher is to fill the gaps that we have seen in Jira. There are plenty. Therefore, this extension will continue to grow.
Options:
Displays a number that denotes the rank within the visible items, excluding done items.
Changes the card colour to a lighter version of the 'bar' - depends on board card colours to be set up.
Adds a hover-over containing the comment used when flagging an item (last comment with text in it).
Adds a hover-over containing the sub-task details including status, name and assignee (sub-tasks must be added to the card layout).
